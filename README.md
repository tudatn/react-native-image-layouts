# react-native-image-layouts

Simple layout component for images

## Installation

```sh
npm install react-native-image-layouts
```

## Usage

```js
import ImageLayouts from "react-native-image-layouts";

// ...

const result = await ImageLayouts.multiply(3, 7);
```
